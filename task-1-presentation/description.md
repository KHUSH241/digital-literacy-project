# Task 1 – Digital Literacy Infographic

**Tool Used:** Canva

This infographic was created using Canva to help batchmates 
understand digital literacy. It covers three sections:
1. What Digital Literacy means
2. Safe Internet Practices
3. Professional Online Presence Tips

One thing I found interesting was how much the layout affects 
how information is received. A text-heavy design looked cluttered, 
so I switched to short bullet points with icons which made it 
much cleaner and easier to read.
